# PsicoHealth — Landing Page de Psicologia

Landing page moderna e responsiva para psicólogo / clínica de psicologia, com estética **dark clássica** inspirada na filosofia e psicologia greco-romana.  
O foco é transmitir **introspecção, sabedoria, equilíbrio emocional e sofisticação**, mantendo uma experiência acolhedora para o paciente.

---

## ✨ Features

- **Design dark clássico** com fundo carvão, detalhes em dourado suave e texturas discretas de mármore/pergaminho.
- **Tipografia elegante**:
  - Títulos com `Cormorant Garamond` (clássico e sofisticado).
  - Corpo com `Crimson Text` (alta legibilidade).
- **Layout responsivo (mobile-first)** utilizando Flexbox e CSS Grid.
- **Navegação fluida**:
  - Rolagem suave (smooth scroll).
  - Menu mobile com botão hambúrguer.
  - Animações sutis ao rolar a página (Intersection Observer).
- **Cards de áreas de atendimento** com páginas detalhadas para:
  - Ansiedade
  - Depressão
  - Burnout
  - Síndrome do Pânico
  - Estresse
  - Baixa Autoestima
  - Problemas de Relacionamento
